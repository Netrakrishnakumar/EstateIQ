About Estate IQ

Learn about our smart house price prediction platform

Project Overview

Estate IQ is a modern web application that leverages the power of IBM Watson Studio AutoAI to provide accurate house price predictions. Our platform uses advanced machine learning models trained on real estate data to help users estimate property values quickly and reliably.

Whether you're a buyer, seller, or real estate professional, Estate IQ provides AI-powered insights to make informed decisions about property investments.

Technologies Used

Backend
  • Python Flask - Web Framework
  • IBM Cloud SDK - Cloud Integration
  • Requests - HTTP Library
  • Python-dotenv - Environment Management
Frontend
  • HTML5 - Markup Language
  • Bootstrap 5 - UI Framework
  • CSS3 - Styling
  • JavaScript - Interactivity

IBM Watson Studio AutoAI Integration

Estate IQ integrates seamlessly with IBM Watson Studio AutoAI, IBM's automated machine learning platform. This integration allows us to:

  • Deploy pre-trained machine learning models for house price prediction
  • Leverage IBM Cloud's secure and scalable infrastructure
  • Use IBM IAM authentication for secure API access
  • Get real-time predictions through REST API endpoints
  • Benefit from IBM's enterprise-grade AI capabilities

The machine learning model is trained on comprehensive real estate datasets and automatically optimized by AutoAI to ensure accurate predictions.
